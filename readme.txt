Описание задачи.
Требуется релизовать графематический анализ текста: в текстовом блоке преобразовать числовое написание даты и времени в прописное.
Описание решения.
Входные данные построчно считываются из файла.
Каждое слово проходит проверку на дату или время.
Если проверка возвращает true, то вызывается соответсвующий метод перевода.
Перевод числового формата времени и даты осуществляется при помощи класса RuleBasedNumberFormat из библиотеки ICU, в конструктор которого передается число и
правило, определяющее падеж.
