Описание задачи.
Требуется релизовать графематический анализ текста: в текстовом блоке преобразовать первое лицо в ретье, а также числовое написание даты и времени в прописное.
Описание решения.
Решение представляет собой автономное кроссплатформенное десктопное приложение.
Приложение на вход принимает фрагмент текста для преобразования и по нажатию кнопки меняет местоимения из первого лица в третье. В ходе преобразования также учитываются склонения глаголов.
Опционально имеется возможность преобразования дат и времени из числового в текстовый формат.
Описание технических особенностей решения.
Входные данные построчно считываются из текстовой области.
Каждое слово проходит проверку на часть речи, а также дату или время.
Если проверка возвращает true, то вызывается соответствующий метод перевода.
Перевод числового формата времени и даты осуществляется при помощи класса RuleBasedNumberFormat из библиотеки ICU, в конструктор которого передается число и правило, определяющее падеж.
Вывод результата осуществляется в web view.
Стек: Java, Java Scipt, stanford-corenlp, google.guava, mvc
Приложение имеет простой интуитивный интерфейс, дружественный неопытному пользователю. Архитектура приложения позволяет расширять функционал с развитием потребностей пользователей.
